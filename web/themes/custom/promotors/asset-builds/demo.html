10 Columns

NOTE: If you are using @include col(6, $componentLayout: 'halves'); with a Base 10 layout, then you must add the $columnBase and change 6 to 5. e.g.

@include col(5, $componentLayout: 'halves', $columnBase: 10);

This is because if you want to do half of base 10, then that is 5 columns, whereas when it is Base 12, then 6 is half.

2 Columns of 5, centered using Skip()

skip() is a mixin that allows you to skip a number of columns. It can be used to create a central layout when the columns itself don't make up the full column-base total e.g. 10 columns in width leaves 2 columns spare when it's a 12 column layout, so skip the first column to maintain central layout.

Using col-left and col-right padding

uses col-left and col-right classes to add inside padding to columns that you may want to style with a space between.
